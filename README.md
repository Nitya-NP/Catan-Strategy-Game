# Catan Strategy Game

This project is an AI-based strategy simulation inspired by the board game Catan, developed in Java as part of a first-year engineering course project and further refined for portfolio use.

The system focuses on object-oriented design principles, modular architecture, and structured decision-making logic to simulate intelligent gameplay between human and AI players.

### Key Features
- AI-driven decision-making for computer players
- Modular and scalable object-oriented design
- Implementation of design patterns (Strategy, Observer, Command)
- Turn-based gameplay simulation with rule-based logic
- Console-based interactive game system

### Technologies Used
Java, Object-Oriented Programming, Design Patterns, AI-inspired logic

### What I Learned
- Designing scalable software systems using OOP principles
- Implementing structured AI logic for game decisions
- Improving code modularity and maintainability through design patterns
